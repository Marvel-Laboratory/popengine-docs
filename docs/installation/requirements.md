# System Requirements

Before installing Last Second Coupon, make sure your Shopware 6 shop meets the following requirements.

## Shopware Version

| Requirement | Minimum |
|---|---|
| Shopware 6 | **6.5 or higher** |

:::tip
We recommend running the latest Shopware 6 version for the best experience and security updates.
:::

## Browser Requirements

### Administration (Popup Editor)

The drag-and-drop popup editor works best in modern browsers:

| Browser | Minimum Version |
|---|---|
| Google Chrome | 90+ |
| Mozilla Firefox | 90+ |
| Microsoft Edge | 90+ |
| Safari | 14+ |

### Storefront (Customer-Facing Popup)

The storefront popup widget is compatible with all modern browsers:

| Browser | Support |
|---|---|
| Chrome, Edge, Firefox, Safari | Full support |
| Mobile Chrome, Mobile Safari | Full support (touch-optimized triggers) |
| Internet Explorer 11 | Not supported |

## Hosting Requirements

Last Second Coupon is a cloud-based app, so there are no special server requirements beyond what Shopware 6 itself needs. The app communicates with our servers to manage popups and coupons.

Your hosting environment needs:

- **Outbound HTTPS access** — Your server must be able to reach our API endpoints
- **JavaScript enabled** — The storefront widget requires JavaScript to be enabled in customer browsers

## Sales Channels

Last Second Coupon works with all Shopware 6 sales channels. You can enable or disable the app for specific sales channels in the [coupon settings](/docs/admin/coupon-settings).
