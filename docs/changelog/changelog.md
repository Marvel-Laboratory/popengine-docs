# Changelog

All notable changes to Last Second Coupon are documented here.

## [1.0.0] - 2026-02-25

### Added
- Initial release
- Drag-and-drop popup editor with GrapesJS
- 6 built-in popup templates (Blank, Exit-Intent Minimal, Holiday Sale, Mother's Day, Minimal B&W, Announcement Bar)
- Multi-popup support with priority-based selection
- Exit-intent detection for desktop (mouse tracking) and mobile (swipe detection)
- Time-on-page trigger with configurable delay
- Percentage and fixed-amount discount types
- Configurable minimum cart value, max discount cap, and coupon expiration
- Countdown timer with customizable duration (1–1,440 minutes)
- Copy-to-clipboard coupon code interaction
- Customer segments: new visitors, returning customers, logged-in customers
- Max uses per customer and cooldown period limits
- Email capture with optional/required mode and newsletter opt-in
- Recovery tab with customizable text, position, and colors
- Display conditions: page targeting, category/product ID targeting, page exclusions
- Country targeting with include/exclude modes
- Monthly impression caps per popup
- Sales channel selection
- Multi-language support (English and German)
- Analytics dashboard with summary metrics and coupon log
- Coupon detail view with config snapshot
- Bulk coupon deletion
- Shadow DOM isolation for theme compatibility
