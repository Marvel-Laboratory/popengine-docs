# Admin Overview

The Last Second Coupon admin panel gives you everything you need to create, manage, and track your exit-intent coupon popups. Access it from your Shopware Administration under **Extensions > Last Second Coupon**.

## Main Sections

### Popup Manager

Create and manage multiple popup designs. Each popup can have its own look, discount rules, and targeting settings. You can run several popups at once with different priorities.

[Learn more about the Popup Manager](/docs/admin/popup-manager)

### Popup Editor

Design your popups with a visual drag-and-drop editor. Choose from ready-made templates or start from scratch. Add countdown timers, coupon code displays, call-to-action buttons, and more.

[Learn more about the Popup Editor](/docs/admin/popup-editor)

### Coupon Settings

Configure discount rules, trigger behavior, usage limits, and email capture settings for each popup. Control exactly when and how coupons are offered to your customers.

[Learn more about Coupon Settings](/docs/admin/coupon-settings)

### Analytics

Track how your popups perform with detailed analytics. See how many coupons were issued, redeemed, and expired. Search and filter through individual coupon records.

[Learn more about Analytics](/docs/admin/analytics)

## Quick Start

1. Open the **Popup Manager** and click **Create Popup**
2. Choose a template that fits your brand
3. Customize the design in the **Popup Editor**
4. Adjust discount and trigger settings
5. Set the popup status to **Active**
6. Monitor results in **Analytics**
