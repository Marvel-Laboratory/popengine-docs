# Popup Manager

The Popup Manager is your central hub for creating and organizing all your coupon popups. You can run multiple popups at the same time, each targeting different pages, customer groups, or countries.

## How It Works

Each popup is an independent unit with its own design, discount rules, display conditions, and targeting. When a customer triggers a popup (for example, by moving to leave your site), the system selects the highest-priority active popup that matches the current page and customer.

## Creating a New Popup

1. Open **Last Second Coupon > Popups** in your Administration
2. Click the **Create Popup** button
3. Choose a starting template:
   - **Blank Canvas** — Start with an empty editor
   - **Exit-Intent Minimal** — Clean single-column popup with headline, coupon code, and countdown
   - **Holiday Sale** — Dark festive theme with a colored accent badge
   - **Mother's Day** — Pastel design with floral accents
   - **Minimal Black & White** — Clean monochrome design
   - **Announcement Bar** — Full-width bar at the top or bottom of the page
4. Give your popup a name (you can change it later)
5. You will be taken to the [Popup Editor](/docs/admin/popup-editor) to customize the design

![Create Popup Dialog](/img/admin/popup-create-dialog.png)

:::tip
Start with a template and customize it. This is faster than building from scratch and ensures your popup has all the essential elements (coupon code, timer, close button).
:::

## Managing Popups

### Popup Status

Each popup has one of three statuses:

| Status | Meaning |
|---|---|
| **Draft** | The popup is being designed and is not visible to customers |
| **Active** | The popup is live on your storefront |
| **Inactive** | The popup is paused and not shown to customers |

To change a popup's status, click the **status badge** next to the popup name in the list.

### Priority

When multiple popups are active, the one with the **highest priority** number is shown first (if it matches the customer's current page and conditions). Set priority in the popup settings.

For example:
- Popup A (priority 10) — Shows on product pages
- Popup B (priority 5) — Shows on all pages

If a customer is on a product page, Popup A is shown because it has higher priority. On the homepage, Popup B would be shown instead.

### Duplicating a Popup

To create a copy of an existing popup:

1. Click the **three-dot menu** next to the popup
2. Select **Duplicate**
3. A copy is created with "(Copy)" added to the name
4. The duplicate starts in **Draft** status

This is useful when you want to create variations of a successful popup for A/B testing or seasonal campaigns.

### Deleting a Popup

1. Click the **three-dot menu** next to the popup
2. Select **Delete**
3. Confirm the deletion

:::caution
Deleting a popup is permanent. Any coupons already generated by the popup will remain in your analytics, but the popup design itself cannot be recovered.
:::

## Multi-Language Support

Each popup supports **English** and **German** content. In the Popup Editor, you can toggle between languages using the **EN / DE** buttons to provide translated popup content.

The storefront widget automatically detects which language the customer's browser or Shopware sales channel is using and shows the matching version.
