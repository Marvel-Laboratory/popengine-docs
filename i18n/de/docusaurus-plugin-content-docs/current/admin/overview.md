# Admin-Übersicht

Das Last Second Coupon Admin-Panel bietet Ihnen alles, was Sie zum Erstellen, Verwalten und Verfolgen Ihrer Exit-Intent-Gutschein-Popups benötigen. Greifen Sie darauf über Ihre Shopware-Administration unter **Erweiterungen > Last Second Coupon** zu.

## Hauptbereiche

### Popup Manager

Erstellen und verwalten Sie mehrere Popup-Designs. Jedes Popup kann sein eigenes Aussehen, eigene Rabattregeln und Targeting-Einstellungen haben. Sie können mehrere Popups gleichzeitig mit unterschiedlichen Prioritäten betreiben.

[Mehr zum Popup Manager erfahren](/docs/admin/popup-manager)

### Popup-Editor

Gestalten Sie Ihre Popups mit einem visuellen Drag-and-Drop-Editor. Wählen Sie aus fertigen Vorlagen oder beginnen Sie von Grund auf. Fügen Sie Countdown-Timer, Gutscheincode-Anzeigen, Call-to-Action-Buttons und mehr hinzu.

[Mehr zum Popup-Editor erfahren](/docs/admin/popup-editor)

### Gutscheineinstellungen

Konfigurieren Sie Rabattregeln, Auslöser-Verhalten, Nutzungslimits und E-Mail-Erfassungseinstellungen für jedes Popup. Kontrollieren Sie genau, wann und wie Gutscheine Ihren Kunden angeboten werden.

[Mehr zu Gutscheineinstellungen erfahren](/docs/admin/coupon-settings)

### Analysen

Verfolgen Sie die Leistung Ihrer Popups mit detaillierten Analysen. Sehen Sie, wie viele Gutscheine ausgestellt, eingelöst und abgelaufen sind. Durchsuchen und filtern Sie einzelne Gutschein-Einträge.

[Mehr zu Analysen erfahren](/docs/admin/analytics)

## Schnellstart

1. Öffnen Sie den **Popup Manager** und klicken Sie auf **Popup erstellen**
2. Wählen Sie eine Vorlage, die zu Ihrer Marke passt
3. Passen Sie das Design im **Popup-Editor** an
4. Stellen Sie Rabatt- und Auslöser-Einstellungen ein
5. Setzen Sie den Popup-Status auf **Aktiv**
6. Überwachen Sie die Ergebnisse in den **Analysen**
