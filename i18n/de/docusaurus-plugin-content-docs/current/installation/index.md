# Installation

Last Second Coupon wird direkt aus dem Shopware Store installiert, genau wie jede andere Shopware 6 App.

## Schritt-für-Schritt-Installation

1. Melden Sie sich in Ihrem **Shopware 6 Administrationsbereich** an
2. Navigieren Sie zu **Erweiterungen > Store**
3. Suchen Sie nach **„Last Second Coupon"**
4. Klicken Sie auf **Erweiterung hinzufügen** und bestätigen Sie die Installation
5. Gehen Sie nach der Installation zu **Erweiterungen > Meine Erweiterungen**
6. Finden Sie **Last Second Coupon** in der Liste und klicken Sie auf **Installieren**
7. Klicken Sie nach der Installation auf **Aktivieren**

![Installation aus dem Shopware Store](/img/admin/installation-store.png)

:::tip
Stellen Sie sicher, dass Ihre Shopware 6 Version die [Systemanforderungen](/docs/installation/requirements) erfüllt, bevor Sie die App installieren.
:::

## Nach der Installation

Nach der Aktivierung wird Last Second Coupon:

- In Ihrer Administrations-Sidebar unter **Erweiterungen** erscheinen
- Automatisch eine Standard-Popup-Vorlage erstellen
- Sinnvolle Standardeinstellungen für Rabatte konfigurieren (10 % Rabatt, 15-Minuten-Timer)

Sie können die App aufrufen, indem Sie in der Administration auf **Last Second Coupon** klicken.

## App aktualisieren

Wenn eine neue Version verfügbar ist:

1. Gehen Sie zu **Erweiterungen > Meine Erweiterungen**
2. Neben Last Second Coupon sehen Sie einen **Aktualisieren**-Button
3. Klicken Sie auf **Aktualisieren** und warten Sie, bis der Vorgang abgeschlossen ist
4. Leeren Sie Ihren Browser-Cache, falls die Admin-Oberfläche veraltet aussieht

:::info
Updates werden automatisch angewendet, ohne dass Ihre Popup-Designs oder Gutscheineinstellungen verloren gehen. Alle vorhandenen Daten bleiben erhalten.
:::

## Deinstallation

Falls Sie die App entfernen müssen:

1. Gehen Sie zu **Erweiterungen > Meine Erweiterungen**
2. Klicken Sie auf das **Drei-Punkte-Menü** neben Last Second Coupon
3. Wählen Sie **Deaktivieren** und dann **Deinstallieren**

:::caution
Die Deinstallation der App entfernt alle Popup-Designs, Gutscheineinstellungen und Analysedaten. Diese Aktion kann nicht rückgängig gemacht werden.
:::
