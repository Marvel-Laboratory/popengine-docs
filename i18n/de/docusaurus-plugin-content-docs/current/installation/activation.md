# Aktivierung & Lizenz

Nach der Installation von Last Second Coupon müssen Sie die App aktivieren, um alle Funktionen nutzen zu können.

## App aktivieren

1. Gehen Sie zu **Erweiterungen > Meine Erweiterungen** in Ihrer Shopware-Administration
2. Finden Sie **Last Second Coupon** in der Liste
3. Klicken Sie auf den **Aktivieren**-Schalter
4. Die App registriert sich bei unseren Servern und richtet Ihren Shop automatisch ein

![App aktivieren](/img/admin/activation-toggle.png)

:::info
Die Aktivierung erfolgt automatisch über das Shopware App-System. Es ist kein Lizenzschlüssel erforderlich — Ihr Shopware Store-Kauf ist Ihre Lizenz.
:::

## Was bei der Aktivierung passiert

Wenn Sie die App aktivieren, passiert Folgendes im Hintergrund:

- Ihr Shop wird beim Last Second Coupon Service registriert
- Ein Standard-Popup-Design wird für Sie erstellt
- Standard-Gutscheineinstellungen werden konfiguriert (10 % Rabatt, 15 Minuten Gültigkeit)
- Das Storefront-Widget-Script wird für Ihre Verkaufskanäle vorbereitet

## Aktivierung überprüfen

So bestätigen Sie, dass die App funktioniert:

1. Öffnen Sie den Bereich **Last Second Coupon** in Ihrer Administration
2. Sie sollten das **Popup Manager** Dashboard sehen
3. Ein Standard-Popup sollte bereits aufgelistet sein
4. Besuchen Sie Ihre Storefront in einem neuen Browser-Tab und versuchen Sie, das Exit-Intent-Popup auszulösen (bewegen Sie die Maus Richtung Schließen-Button des Browsers)
