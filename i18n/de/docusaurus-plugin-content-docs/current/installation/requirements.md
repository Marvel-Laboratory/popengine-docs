# Systemanforderungen

Stellen Sie vor der Installation von Last Second Coupon sicher, dass Ihr Shopware 6 Shop die folgenden Anforderungen erfüllt.

## Shopware-Version

| Anforderung | Minimum |
|---|---|
| Shopware 6 | **6.5 oder höher** |

:::tip
Wir empfehlen, die neueste Shopware 6 Version zu verwenden, um die beste Erfahrung und Sicherheitsupdates zu erhalten.
:::

## Browser-Anforderungen

### Administration (Popup-Editor)

Der Drag-and-Drop-Popup-Editor funktioniert am besten in modernen Browsern:

| Browser | Mindestversion |
|---|---|
| Google Chrome | 90+ |
| Mozilla Firefox | 90+ |
| Microsoft Edge | 90+ |
| Safari | 14+ |

### Storefront (Kunden-Popup)

Das Storefront-Popup-Widget ist mit allen modernen Browsern kompatibel:

| Browser | Unterstützung |
|---|---|
| Chrome, Edge, Firefox, Safari | Volle Unterstützung |
| Mobiles Chrome, Mobiles Safari | Volle Unterstützung (touch-optimierte Auslöser) |
| Internet Explorer 11 | Nicht unterstützt |

## Hosting-Anforderungen

Last Second Coupon ist eine cloudbasierte App, daher gibt es keine besonderen Serveranforderungen über die Anforderungen von Shopware 6 hinaus. Die App kommuniziert mit unseren Servern zur Verwaltung von Popups und Gutscheinen.

Ihre Hosting-Umgebung benötigt:

- **Ausgehender HTTPS-Zugang** — Ihr Server muss unsere API-Endpunkte erreichen können
- **JavaScript aktiviert** — Das Storefront-Widget erfordert, dass JavaScript in den Kundenbrowsern aktiviert ist

## Verkaufskanäle

Last Second Coupon funktioniert mit allen Shopware 6 Verkaufskanälen. Sie können die App für bestimmte Verkaufskanäle in den [Gutscheineinstellungen](/docs/admin/coupon-settings) aktivieren oder deaktivieren.
